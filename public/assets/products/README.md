# Ассеты продуктов

Сюда положи рендеры всех 4 продуктов в высоком разрешении (4K+).

## Текущие файлы

- `hvs-nu.png` — Насосная станция водоснабжения
- `hvs-nu-red2.png` — Насосная станция пожаротушения (красный шкаф)
- `vpu.png` — Установка водоподготовки

## Нужно добавить

- `heat.png` — Блочный тепловой пункт (пока нет)

## Требования к файлам

- **Разрешение:** 4K минимум (3840×2160)
- **Фон:** чёрный сплошной (#0A0A0A) или прозрачный PNG
- **Формат:** PNG с прозрачностью предпочтительно, либо JPG
- **Освещение:** драматичное, контрастное, с бликами на стали
- **Цветокор:** холодный тон, подчёркнутая текстура металла

## Что будет добавлено в ходе работы

- `hvs-nu@2x.webp` — оптимизированная WebP-версия
- `hvs-nu@2x.avif` — оптимизированная AVIF-версия
- `hvs-nu-thumb.jpg` — превью для карточки
- `hvs-nu.glb` — 3D-модель
- `hvs-nu-360/` — папка с 12 кадрами для mobile 360° viewer

## Обработка

Все изображения будут сжиматься через Next/Image автоматически. Исходники сохраняются как есть — не ресайзить вручную.
