# ТЗ на создание сайта ANHEL

## 0. Контекст и задача

Редизайн корпоративного сайта **ANHEL** ([anhelspb.com](https://anhelspb.com/main/)) — производителя инженерного оборудования в Санкт-Петербурге.

**Проблема:** текущий сайт выглядит как каталог 2022 года — плоско, без эмоции, без «вау».

**Цель:** сделать сайт уровня terminal-industries.com и mont-fort.com — кинематографичный, с глубоким ощущением инженерной мощи, технологичности и премиальности.

## Позиционирование

ANHEL® — проектирование и производство модульного инженерного оборудования высшего класса. Не «поставщик», а **инженерный партнёр**, создающий сердце инженерных систем для жилых комплексов, бизнес-центров, промышленных объектов и инфраструктуры.

## Вау-эффект (ключевые ощущения)

- Сталь, вода, давление, точность
- Кинематографичность (как у Apple product page)
- Scroll-driven storytelling (как Terminal)
- Горизонтальные и цифровые счётчики (как Mont-Fort)
- 3D-взаимодействие с продуктом (вращение, разрез, подсветка узлов)

---

## 1. Стек

- **Frontend:** Next.js 14 (App Router) + React 18 + TypeScript
- **Стили:** Tailwind CSS + CSS Variables
- **Анимации:** Framer Motion + GSAP (ScrollTrigger, SplitText) + Lenis (smooth scroll)
- **3D:** Three.js + React Three Fiber + Drei
- **CMS:** Sanity (рекомендуется) или Strapi
- **Деплой:** Vercel (рекомендуется) или собственный VPS
- **Формы:** Nodemailer/Resend + интеграция с Telegram-ботом отдела продаж
- **Аналитика:** Яндекс.Метрика
- **Защита:** reCAPTCHA v3

## 2. Структура сайта

```
/                           — главная (hero + 4 продукта + применение + о компании + контакты)
/products                   — каталог (4 карточки с превью-анимацией)
/products/water-supply      — Насосная станция водоснабжения
/products/fire-fighting     — Насосная станция пожаротушения
/products/water-treatment   — Установка водоподготовки
/products/heat-unit         — Блочный индивидуальный тепловой пункт (БИТП)
/applications               — Применение (жилой дом / промышленность / инфраструктура / ТРЦ)
/about                      — О компании, производство, команда
/documentation              — Каталоги, опросные листы, сертификаты
/contacts                   — Контакты + форма + карта
```

## 3. Визуальный язык

### Палитра

| Цвет | HEX | Назначение |
|------|-----|------------|
| Primary | `#0A0A0A` | глубокий угольно-чёрный фон |
| Secondary | `#F5F5F3` | молочный, для светлых секций |
| Steel 1 | `#C0C8D0` | светлая сталь |
| Steel 2 | `#8A94A0` | тёмная сталь |
| Accent red | `#D72638` | пожаротушение |
| Accent blue | `#1E6FD9` | водоснабжение |
| Hairline | `rgba(255,255,255,0.08)` | тонкие разделители |

### Типографика

- **Заголовки:** Neue Haas Grotesk Display или PP Neue Montreal (fallback: Inter Tight). Hero: 120-180px desktop, tracking -2%, line-height 0.95
- **Основной текст:** Inter или Söhne. 16-18px, line-height 1.5
- **Моно-акцент:** JetBrains Mono — для счётчиков, тех-подписей (`01 / 04`, `MODEL HVS-NU`, `Q: 120 м³/ч`)

### Сетка

- 12-колоночная, gutter 24px
- max-width 1440px, padding 48-120px
- Секции высотой 100vh и выше для scrollytelling

## 5. Страница продукта

Подробное описание в `docs/04_product_page.md`. Основное:

- 3D-модель продукта с ручным вращением (drag), разрезом, подсвеченными узлами

- Сертификаты, кейсы, форма запроса КП

## 6. Анимации

Полный список в `docs/05_animations.md`. Ключевое:

- **Lenis smooth scroll** — плавная инерция
- **Кастомный курсор** с магнитными кнопками
- **Page transitions** через чёрный занавес
- **Text reveal** — по словам, снизу вверх
- **Number counters** — одометр-эффект
- **Horizontal pinned sections** — блок продуктов и применений
- **Image parallax** — крупные изображения с разной скоростью
- **prefers-reduced-motion** — полное отключение для пользователей с чувствительностью

## 7. Мобильная версия

- Горизонтальные pin-секции → вертикальный слайдер-карусель
- 3D-модели → 360° свайп из PNG-рендеров (8-12 кадров)
- Шрифты hero: 56-72px
- Hamburger с полноэкранным меню
- Отключение parallax на тач-устройствах

## 8. Бэкенд и CMS

- **Sanity Headless CMS** — админка для контент-менеджера Anhel
- **Модели контента:** Product, Application, Case, Article, Document
- **Формы:** отправка на `info@anhelspb.com` + Telegram-уведомление
- **Аналитика:** Яндекс.Метрика с events (скачивание PDF, запрос КП, звонок)
- **Защита:** reCAPTCHA v3

## 9. Производительность и SEO

Подробно в `docs/06_performance_seo.md`. Цели:

- **Lighthouse:** Performance 90+, Accessibility 95+, SEO 100
- **LCP** < 2.5s, **CLS** < 0.1
- Видео WebM + MP4 fallback, ≤3 МБ на hero
- Изображения через Next/Image, AVIF + WebP
- 3D-модели в GLB + Draco-сжатие, ленивая загрузка
- SEO-теги под каждый продукт, Schema.org Product, sitemap.xml, Open Graph
- **Локализация:** RU (основная), EN — вторая очередь

## 10. Доступность (a11y)

- Полная навигация с клавиатуры
- Контраст текста ≥ 4.5:1
- `prefers-reduced-motion` отключает параллакс и видео
- aria-labels на иконках-кнопках
- Alt-тексты на всех изображениях
