# Референсы и прототипы

Сюда кладутся скриншоты сайтов-референсов и прототипы ANHEL.

## Должно быть в этой папке

### Референсы (сохранить сюда)

- `terminal-industries-hero.png` — скриншот hero
- `terminal-industries-horizontal.png` — горизонтальный pin-скролл
- `mont-fort-hero.png`
- `mont-fort-counters.png` — анимированные цифры
- `mont-fort-smooth-scroll.mp4` — запись плавного смуз-скролла

### Прототип ANHEL hero

- `anhel-hero-prototype.png` — прототип от Claude (сделан в чате)

### Apple / другие премиум-референсы

- `apple-mac-pro-3d.png` — интерактивные 3D-модели продуктов
- `resend-hero.png` — пример современного B2B
- `linear-app.png` — минималистичная типографика

## Как сохранять

**Для скриншотов:** Cmd+Shift+4 (Mac) или Snipping Tool (Windows), сохранить как PNG.

**Для записи взаимодействий:** Cmd+Shift+5 (Mac) → запись экрана, формат MP4.

## Зачем нужны

- Команде разработки — чтобы понять эталон качества
- Дизайнеру — как база для его работы
- Заказчику — визуализация того что получится
