## Содержательные решения (мои, требуют подтверждения)

### 1. Серийное обозначение: `HVS-NU` (без суффикса)

Использую ту же серию, что у firefighting. Логика: унифицированный корпус, различаются жидкостью (огневая/питьевая) и accent цветом. Это working draft.

**Альтернатива** если заказчик захочет разделить: `HVS-NU-W` (Water Supply) и `HVS-NU-F` (Firefighting). Или совсем другие обозначения (`HVS-WS`, `HVS-FW`).

**Флаг:** подтвердить финальное обозначение.

### 2. ТТХ — значения extrapolated

Sandbox allowlist не включает `mfmc.ru` и `anhelspb.com` — не могу забрать описание и таблицу ТТХ 1-в-1. Взял общие инженерные нормы для насосных станций ХВС/ГВС для жилых/коммерческих объектов:

| Параметр | Значение |
|---|---|
| Количество насосов | от 2 до 6 |
| Производительность | 5 – 500 м³/ч |
| Напор | до 160 м.в.ст. |
| Макс. температура жидкости | 70 °С (стандарт для ГВС) |
| Сетевое напряжение | 3 × 380 В |
| Мощность одного насоса | 0,37 – 200 кВт |
| Тип регулирования | частотное / релейное |
| Срок службы | не менее 10 лет |

**Флаг:** нужен review от инженера ANHEL® — значения могут быть слишком широкие или узкие для фактической линейки. Особенно проверить верхнюю границу производительности (500 м³/ч — крупные объекты).

### 3. Применение — 6 категорий

Идентичны firefighting по структуре, адаптирован текст и порядок:
1. Жилые комплексы
2. Бизнес-центры
3. Торговые центры
4. Гостиницы
5. Промышленные объекты
6. Инфраструктурные объекты

Для водоснабжения добавлены коммерческие примеры (apart-complexes, office centres, ритейл-парки и т.д.) — рынок шире чем для firefighting.

### 4. Кейсы — placeholders

Для водоснабжения нет утверждённого списка implemented-projects. Добавил 3 placeholder-карточки с нейтральными названиями («ЖК Пример №1», «ЖК Пример №2», «Бизнес-центр класса А»). Equipment — «ANHEL® HVS-NU».

**Флаг:** заменить на реальные объекты когда заказчик передаст список.

### 5. Бренды — те же что у firefighting

Насосы (7): AquaDeus, CNP, Wilo, Lowara, Leo, ГМС, KQ. Компоненты (9): DEKraft, Unitronics, IEK, КЭАЗ, EKF, CHINT, DKC, Weintek, ТИТАН Контрол. Обоснование: OEM-завод использует одинаковый пул комплектующих для всех product lines.

Добавил у Wilo дополнительную серию `MVIE` (centrifugal vertical, типичная для ХВС) и у Leo — `APM` (circulation pumps).

**Флаг:** проверить актуальность списка брендов для водоснабжения специфически — возможно для ХВС/ГВС есть доп. бренды, которых нет в firefighting линейке.

### 6. Преимущества — 9 пунктов

Основа та же, что у firefighting. Два пункта адаптированы:
- **02 «Индивидуальное исполнение»** — текст про гидравлические параметры вместо ТЗ по пожаротушению
- **05 «Круглосуточный режим»** — вместо «Надёжность» — делает акцент на непрерывной работе (водоснабжение это 24/7 в отличие от аварийного пожарного запуска)

### 7. Документы — placeholder PDF

Четыре карточки с href `/docs/water-supply/*.pdf` — файлов нет в `public/`, клик даст 404. Gap из Group 1 — реальные PDF ещё не готовы у заказчика.

### 8. Галерея — skeletons

Те же 8 placeholder-плиток с описанием как у firefighting, с адаптацией подписей (например «Насосные агрегаты ХВС на сборке»). Реальные фото появятся после shooting-session.
